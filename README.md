# BeadMaker

BeadMaker is a modern Android application designed for beadweaving pattern designers. It provides a flexible grid-based editor with support for professional stitches, template overlays, and a beautiful UI.

## ✨ Features

- **Professional Stitch Support**:
  - **Square Grid**: Perfect for lettering and pixel art.
  - **Peyote (1-Drop, 2-Drop, 3-Drop)**: Staggered patterns for bracelets and tapestries.
  - **Brick Stitch**: Offset horizontal rows for shaped jewelry like earrings.
- **Advanced Editor Tools**:
  - **Template Overlay**: Import images or take a photo to use as a transparent guide behind your grid.
  - **Zoom & Pan**: Easily navigate large grids with intuitive touch gestures.
  - **Color Palette**: Customizable palette to paint your designs.
  - **Eraser Tool**: Quickly fix mistakes.
- **Dynamic Theming**:
  - **Pastel Light**: A soft, warm theme for daytime designing.
  - **Tokyo Night Moon**: A high-contrast dark theme inspired by the futuristic midnight sky of Tokyo.
- **Flexible Grid Settings**: Custom width/height and grid clearing options.

## 🎨 Design

The app is built using **Jetpack Compose** and follows **Material 3** design principles. It features a clean, focused interface with a dedicated "Tools" menu to keep the workspace clutter-free.

## 🚀 Getting Started

### Prerequisites
- Android Studio Ladybug (or newer)
- Android SDK 34+
- JDK 17+

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/makiritare/BeadMaker.git
   ```
2. Open the project in Android Studio.
3. Sync Project with Gradle Files.
4. Run the app on an emulator or physical device.
